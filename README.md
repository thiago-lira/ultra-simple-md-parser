# Ultra Simple Markdown Parser

A very simple markdown parser.

## Getting started

**Installing**:

```shell
npm i ultra-simple-md-parser -save
```

**Usage:**
```js
import mdParser
```

## API

**Headings**

**H1**
```javascript
mdParser.h1(myMdText)
```

**H2**
```javascript
mdParser.h2(myMdText)
```

**H3**
```javascript
mdParser.h3(myMdText)
```

**H4**
```javascript
mdParser.h4(myMdText)
```

**H5**
```javascript
mdParser.h1(myMdText)
```
**H6**
```javascript
mdParser.h6(myMdText)
```
